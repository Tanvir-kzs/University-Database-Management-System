# University Management System using Oracle SQL & PL/SQL

## Description
This project implements a University Management System using Oracle Database. It demonstrates database design, SQL queries, and PL/SQL programming concepts.

## Features
- Student Management
- Department Management
- Instructor Management
- Course Management
- Classroom Management
- Semester Management
- Enrollment System

## Technologies Used
- Oracle Database
- SQL
- PL/SQL

## Tables
1. Department
2. Student
3. Instructor
4. Course
5. Enrollment
6. Classroom
7. Semester

## PL/SQL Operations
- Anonymous Block
- Procedure
- Function
- Cursor
- Trigger
- Exception Handling
